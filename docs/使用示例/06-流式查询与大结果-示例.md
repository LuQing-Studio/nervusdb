# 06 · 流式查询与大结果

```ts
import { SynapseDB } from 'synapsedb';

const db = await SynapseDB.open('big.synapsedb');

// 批次 1000 条返回（避免一次性加载大结果集）
for await (const batch of db.streamFacts({ predicate: 'KNOWS' }, 1000)) {
  // 逐批处理
  console.log('got', batch.length);
}

await db.close();
```

要点

- `streamFacts(criteria, batchSize)`：按批异步生成器
- 结合 SSE/WebSocket 可渐进下发到前端
